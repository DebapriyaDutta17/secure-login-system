🔐 Secure Login System

A secure web-based authentication system built using Flask, SQLite, and Bcrypt. This project demonstrates essential cybersecurity practices including password hashing, session management, input validation, and protection against SQL Injection attacks.

🚀 Features

User Registration & Login
Password Hashing with Bcrypt
Input Validation
SQL Injection Protection
Session-Based Authentication
Secure Logout Functionality
Responsive and Simple UI
Optional Two-Factor Authentication (2FA)
